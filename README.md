This incredible project is a imageboard clone, similar to Safebooru and other tag-based anime picture websites. The goal of this project is to learn about storing a lot of images using Cloudflare's R2 and easly finding them using tags thanks to data stored in PSQL database. The application itself is written in Express.js with EJS used for templating.

## Installation and running
`npm i`
`npm run dev`
